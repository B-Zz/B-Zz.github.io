从零开始的Go(5)-字符串

Oct 19, 2022
阅读时长: 3 分钟

字符串

在Go中，字符串是 UTF-8 字符的一个序列，一个字符可能占用1个字节(ASCII码)，或是2~4个字节。

定义

通过以下两种形式定义字符串：

1
2

var s1 string
s2 := "Hello!"

索引与切片

索引

可以将一个字符串看成数组，来获取其某一字节上的值。类似数组，string[0]对应第一个字节。

1
2
3
4

s1 := "hello"
fmt.Println(s1[0])	//结果为104，对于h的ASCII码
fmt.Println(s1[3])	//结果为108，对于l的ASCII码
fmt.Println(string(s1[3]))	//加上string()函数,则输出字符

而对于中文等多字节字符，并不可以直接使用索引来得出各位字符，只能得到某个字节的值，需要先转换为rune切片：

1
2
3

s2 := "你好！"
fmt.Println(([]rune(s2))[0])	//结果为20320，为4f60的十进制表示形式，为'你'的UTF-8编码
fmt.Println(string(([]rune(s2))[0]))	//结果为"你"

切片

通过以下方式来获取字符串的某几个连续字节上的值：

1
2
3
4
5

s1 := "hello!"
fmt.Println(s1[0:5])	// "hello"	遵循向前取整的规则，取到索引为4的字节
fmt.Println(s1[:5])	// "hello"	前一个数为空，即从0开始
fmt.Println(s1[4:])	// "o!"	后一个数为空，即到最后一个字符为止
fmt.Println(s1[:]) 	// "hello!"	若两个字符都为空，则取整个字符串

同样的，对与中文字符，需要先转化为rune切片：

1
2
3

s2 := "你好！"
fmt.Println(([]rune(s2))[0:2])	//结果为[20320 22909]，为'你'与'好'的UTF-8编码
fmt.Println(string(([]rune(s2))[0:2]))	//加上string()函数，则输出"你好"

编辑

在Go中，无法直接对string的某一位上的值进行改变，需要先进行类型的转化。

运行以下命令会出错：

1
2

s1 := "hello!"
s1[0] = '1'

如果是单一的ASCII码组成的字符串，则可以转化成[]byte类型，而含有多字节字符的则需转化位[]rune类型。

1
2
3
4
5
6
7
8
9

s1 := "hello!"
s1_ := []byte(s1)	//转化成[]byte类型
s1_[0] = 'H'
fmt.Println(string(s1_))	//结果为"Hello!"

s2 := "你好！"
s2_ := []rune(s2)	//转化成[]rune类型
s2_[0] = "您"
fmt.Println(string(s2_))	//结果为"您好！"

拼接与比较

拼接

使用+对字符串进行拼接（当然可以使用+=），如：

1
2
3

s1 := "Hello "
s2 := "World!"
s3 := s1+s2		//s3结果为"Hello World!"

比较

字符串的比较是按字节比较，从第一个字节开始比较，大的为字符串的第一个字节大的字符串，如果第一个字节一样大，则顺次比较。

1
2
3

s1 := "ab"
s2 := "123"
fmt.Println(s1 > s2) 	//结果为true

求长度

可以通过len()函数来获取字符串的长度，但是这个函数实际上求的是字符串的字节长度。

可以通过以下代码来看出：

1
2
3
4
5

s1 := "Hello!"
fmt.Println(len(s1))	//得出的结果为6，一个ASCII字符占1个字节

s2 :=  "你好！"
fmt.Println(len(s2))	//得出的结果为9，在UTF-8中，每一个中文及全角字符占3个字节

那要如何计算出除ASCII码之外字符的实际长度呢？也需转化为rune切片：

1
2

s2 :=  "你好！"
fmt.Println(len([]rune(s2)))	//得出结果为3

相关包

strings包

strings包为go提供了一系列对字符串进行操作的函数。

前缀与后缀

1
2

strings.HasPrefix(s, prefix string) bool	//判断前缀，若对与字符串s来说，前缀为prefix，则返回true，反之则返回false
strings.HasSuffix(s, suffix string) bool	//判断后缀，同判断前缀

示例：

1
2

s1 := "hello"
fmt.Println(strings.HasPrefix(s1,"h"))	//结果为true

包含

strings.Contains(s, substr string) bool

判断字符串s是否包含substr部分，若包含，返回true，反之则返回false

示例：

1
2
3

s1 := "hello"
fmt.Println(strings.Contains(s1,"hell"))	//结果为true
fmt.Println(strings.Contains(s1,"helll"))	//结果为false

获取某字符串在另一字符串中位置(索引)

返回为str在字符串s中的第一个索引位置，若str为字符串，则返回第一个字符对应的索引位置。

1
2
3

strings.Index(s, str string) int	
strings.LastIndex(s, str string) int	//规则同Index，但是返回最后一个符合的索引位置
strings.IndexRune(s string, r rune) int		//适用于非ASCII字符串

示例：

1
2
3
4

s1 := "hello"
fmt.Println(strings.Index(s1, "l"))	//结果为2
fmt.Println(strings.Index(s1, "el"))	//结果为1
fmt.Println(strings.LastIndex(s1, "l"))	//结果为3

出现次数统计

strings.Count(s, str string) int

统计str在字符串s中出现的次数，并返回。

示例：

1
2
3

s1 := "hello"
fmt.Println(strings.Count(s1, "l"))	//结果为2
fmt.Println(strings.Count(s1, "e"))	//结果为1

重复字符串

重复count次字符串s，并返回。

1

strings.Repeat(s, count int) string

示例：

1
2

s1 := "hello "
fmt.Println(strings.Repeat(s1, 3))

从字符串中读取内容

函数 strings.NewReader() 用于生成一个 Reader 并读取字符串中的内容，然后返回指向该 Reader 的指针，从其它类型读取内容的函数还有：

Read()从[]byte中读取内容。 ReadByte() 和 ReadRune()从字符串中读取下一个 byte 或者 rune。

字符串替换

strings.Replace(str, old, new string, n int) string

将字符串str中的前 n 个字符串old替换为字符串new，并返回一个新的字符串。如果n = -1则替换所有字符串old为字符串new

示例：

1
2
3

s1 := "hello"
fmt.Println(strings.Replace(s1,"l","o",1))	//heolo
fmt.Println(strings.Replace(s1,"l","o",-1))	//heooo

修剪字符串

使用 strings.TrimSpace() 来剔除字符串开头和结尾的空白符号

如果想要剔除指定字符，则可以使用 strings.Trim(s, "cut") 来将开头和结尾的 cut 部分去除掉

如果只想剔除开头或者结尾的字符串，则可以使用 strings.TrimLeft 或者 strings.TrimRight 来实现

示例：

1
2
3
4
5
6

s1 := " hello "	
s2 := "hellhe"	
fmt.Println(strings.TrimSpace(s1))	//hello
fmt.Println(strings.Trim(s2, "he"))	//ll
fmt.Println(strings.TrimLeft(s2, "he"))	//llhe
fmt.Println(strings.TrimRight(s2, "he"))	//hell

大小写转换

1
2

strings.ToLower(s) string	//将字符串s中的所有大写字母转换为小写字母
strings.ToUpper(s) string	//将字符串s中的所有小写字母转换为大写字母

示例：

1
2
3
4

s1 := "hello!"
fmt.Println(strings.ToUpper(s1))
s2 := "HeLLO!"
fmt.Println(strings.ToLower(s2))

strconv包

strconv包为go提供了许多字符串与其他类型转换的函数。

数字转换字符串

strconv.Itoa(i int) string

返回数字 i 所表示的字符串类型的十进制数

strconv.FormatFloat(f float64, fmt byte, prec int, bitSize int) string

将 64 位浮点型的数字f转换为字符串并返回，其中 fmt 表示格式（其值可以是 ‘b’、’e’、‘f’ 或 ‘g’），prec 表示精度，bitSize 则使用 32 表示 float32，用 64 表示 float64

字符串转换数字

转换为整形

strconv.Atoi(s string) (i int, err error)

将数字字符串s转化为整数i，并返回，如果产生错误，返回err

示例：

1
2
3
4
5
6

s1 := "123"
a, err := strconv.Atoi(s1) 
// 若不需要返回错误，可以直接这样写：
//a, _ := strconv.Atoi(s1) 
fmt.Println(a)
fmt.Println(err)

转换为浮点型

strconv.ParseFloat(s string, bitSize int) (f float64, err error)

将数字字符串(可以含小数)s转换为浮点型f，如果产生错误，返回err，bitSize可以位32或64，表示float32或float64

其他

1

strconv.IntSize	//获取程序运行的操作系统平台下 int 类型所占的位数
