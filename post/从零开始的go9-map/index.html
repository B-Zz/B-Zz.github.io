从零开始的Go(9)-Map

Oct 28, 2022
阅读时长: 1 分钟

定义

可以使用map()来声明一个map。

1

var identifier map[keytype]valuetype	//只是声明了一个map，但是没有分配内存，不能用来存放键值。

一个map是无序的键值对的集合，类似与python中的dictionary，可以动态增加，未初始化的map值为nil。

在map中，key可以为float、int、string等类型，但不能为切片与结构体(含有数组切片的结构体不能作为 key，只包含内建类型的 struct 是可以作为 key 的)，可以为指针与接口。

而value可以为任意类型(但是使用空接口时，需要先做一次类型断言)。

使用len()可以获取map中键值对的数量。

初始化

在go中，map是引用类型，未初始化的值为nil，使用以下方式来初始化map：

1
2
3
4
5

var map1 map[string]string 
map1 = make(map[string]string)

var map2 map[string]string
map2 = map[string]string{"key1":"1", "key2":"2"}

也可以将声明与初始化写在一起：

1
2

map1 := make(map[string]string)	//声明+分配内存
map2 := map[string]string{"key1":"1", "key2":"2"}	//声明+初始化

使用

添加元素

格式：

1

identifier [newkey] = value	//需要先初始化

删除元素

使用delete()函数来删除map中的元素：

1

delete(identifier, "key")

获取key对应的value

格式：

1
2

a,b = identifier[key]	//将map identifier的key值对应的value赋给变量a
// b将返回一个布尔值，若key存在则为true，反之则为false，若不需要判断，可以省略

for-range的使用

格式：

1
2
3

for key, value := range map1 {
	fmt.Printf("key is: %d\n", key)
}

第一个返回值key为map的键值，第二个返回值value为键值对应的值。但是，map是一个无序集合，所以返回的key的顺序并无法确定。
